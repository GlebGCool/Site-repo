Remote Admin

Цель проекта

Поработать с сетями через C#

Стек технологий

C#, Sockets, SQLite

Суть и функционал

Ну короче есть команды вот основное
msg - отправить текст на экран.
lock - заблокировать компьютер.
status - узнать время неактивности ПК.
rb / sd - перезагрузка / выключение.
Доп функционал
close - прервать связь с определённым клиентом.
show - показать окно программы для определённого клиента.
hide - скрыть окно программы для определённого клиента.
lscli - получить список подключённых клиентов.
help - получить список команд.
